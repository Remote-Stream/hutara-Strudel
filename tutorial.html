شروع سریع در 3 دقیقه

CyberSynth Strudel یک استودیوی قدرتمند برای ساخت موسیقی با کد است. در 3 دقیقه اولین موسیقی خود را بسازید!

گام 1: اولین صدا

در ویرایشگر کد بنویسید:

sound("bd hh sd hh")

دکمه Evaluate را بزنید یا Ctrl+Enter

گام 2: لایه‌بندی

چند صدا را ترکیب کنید:

stack( sound("bd*2"), sound("hh*8").gain(0.5) )

گام 3: افکت‌ها

افکت اضافه کنید:

sound("bd hh sd hh") .room(0.5) .delay(0.3)
برای توقف صدا، دکمه Stop را بزنید یا hush() را اجرا کنید.

آشنایی با رابط کاربری

CyberSynth Strudel از چندین پنل تخصصی تشکیل شده که هر کدام برای کار خاصی طراحی شده‌اند:

نوار کنترل

  • Play/Pause: پخش/توقف موسیقی
  • Stop: توقف کامل صدا
  • Undo/Redo: برگشت/جلو رفتن در تاریخچه

پنل‌های اصلی

  • Sequencer: ساخت ریتم گام به گام
  • Code Editor: نوشتن کد Strudel
  • Synth Controls: تنظیم صدا
  • Pattern Bank: ذخیره الگوها

ذخیره‌سازی

  • ذخیره خودکار در مرورگر
  • صادرات/واردات JSON
  • بانک پترن شخصی

مبانی Strudel

Strudel یک زبان برنامه‌نویسی برای موسیقی الگوریتمی است که در مرورگر اجرا می‌شود.

مفاهیم کلیدی

الگوها (Patterns)

الگوها واحدهای اصلی Strudel هستند که در زمان تکرار می‌شوند:

// الگوی ساده sound("bd hh sd hh") // الگوی تکراری sound("bd*4") // چهار بار تکرار // الگوی تصادفی sound("{bd sd cp}") // انتخاب تصادفی

صداها (Sounds)

از نمونه‌های صوتی از پیش تعریف شده استفاده کنید:

  • bd - Bass Drum (کیک)
  • sd - Snare Drum (اسنر)
  • hh - Hi-Hat (های‌هت)
  • cp - Clap (کف)
  • bass - Bass (بیس)

ترکیب صداها:

// استفاده از شماره نمونه sound("bd:1 sd:2 hh:3") // ترکیب با ویرگول (همزمان) sound("bd, hh*2, ~ sd")

نت‌ها (Notes)

برای ملودی‌ها از نت‌های موسیقی استفاده کنید:

// نت‌های ساده note("c4 d4 e4 f4 g4") // با سینتی‌سایزر note("c4 e4 g4").sound("sawtooth") // آکوردها note("[c4,e4,g4]").sound("piano")

نحو کدنویسی پیشرفته

عملگرهای اصلی

تکرار و سرعت

// تکرار با * sound("bd*4 hh*8") // سرعت sound("bd hh").fast(2) // دو برابر سریع‌تر sound("bd hh").slow(2) // نصف سرعت // ترکیب sound("bd*2 hh*4").fast(1.5)

الگوهای پیچیده

// گروه‌بندی با [] sound("bd [hh hh] sd [hh hh hh]") // انتخاب با < > sound("bd ") // هر سیکل یکی // تصادفی با { } sound("bd {hh sd cp}") // انتخاب تصادفی // ترکیبی sound("[bd*2, {~ hh}] ")

توابع تبدیل

// معکوس کردن sound("bd hh sd cp").rev() // جابجایی sound("bd hh sd cp").rotate(1) // اقلیدسی sound("bd").euclid(3,8) // 3 ضربه در 8 گام // ترکیب با jux sound("bd hh").jux(rev)

کار با سکوئنسر پیشرفته

سکوئنسر برای ساخت ریتم‌های دقیق گام به گام طراحی شده است.

ترک‌های درام

  • KICK: ضربات بیس (کیک)
  • SNARE: اسنر درام
  • HI-HAT: های‌هت
  • CLAP: کف (کلپ)

کنترل‌های ترک

  • Mute: بی‌صدا کردن ترک
  • Solo: فقط این ترک
  • Steps: تعداد گام‌ها (4/8/16)
  • Randomize: الگوی تصادفی

تولید کد

  • Generate: تبدیل به کد
  • Generate All: همه ترک‌ها
  • Clear All: پاک کردن

مثال: ساخت ریتم 4/4

1. در ترک KICK گام‌های 1، 5، 9، 13 را فعال کنید

2. در ترک HI-HAT همه گام‌ها را فعال کنید

3. در ترک SNARE گام‌های 5 و 13 را فعال کنید

4. دکمه Generate All را بزنید

// کد تولید شده: stack( s("bd*16").struct("x~~~x~~~x~~~x~~~"), s("hh*16").struct("xxxxxxxxxxxxxxxx"), s("sd*16").struct("~~~~x~~~~~~~x~~~") )

ویرایشگر کد زنده

ویرایشگر کد قلب CyberSynth است که امکان نوشتن و اجرای فوری کد را فراهم می‌کند.

ویژگی‌ها

اجرای زنده

کد را بنویسید و با Evaluate یا Ctrl+Enter اجرا کنید

ذخیره‌سازی

کد در مرورگر ذخیره می‌شود و با Save/Load قابل مدیریت است

تاریخچه

با Undo/Redo می‌توانید تغییرات را برگردانید

از hush() در ابتدای کد استفاده کنید تا صداهای قبلی متوقف شوند.

کنترل‌های سینتی‌سایزر

پارامترهای صوتی را به صورت بصری تنظیم کنید.

فیلترها

  • LPF (Low Pass Filter): فیلتر پایین‌گذر - فرکانس‌های بالا را حذف می‌کند
  • LPQ (Resonance): رزونانس - تشدید فرکانس فیلتر
// استفاده از فیلتر sound("bass*4") .lpf(800) // فرکانس فیلتر .lpq(5) // رزونانس // فیلتر متحرک sound("bass*4") .lpf(sine.range(200,2000).slow(4))

افکت‌ها

  • Room (Reverb): ریورب - حس فضا
  • Delay: تاخیر - اکوی صدا
// اضافه کردن افکت‌ها sound("bd hh sd hh") .room(0.5) // 50% ریورب .delay(0.3) // 30% دیلی .gain(0.8) // حجم صدا
دکمه Apply to Code پارامترها را به کد فعلی اضافه می‌کند.

بانک پترن

الگوهای خود را ذخیره و مدیریت کنید.

عملیات پترن

ذخیره پترن

  1. الگو را در ویرایشگر بسازید
  2. نام پترن را وارد کنید
  3. Save Pattern را بزنید

پخش و ترکیب

  • Play: پخش مستقیم
  • Mix: ترکیب با کد فعلی
  • Load: بارگذاری کامل

صادرات/واردات

  • Export All: ذخیره JSON
  • Import: بارگذاری از فایل
  • پشتیبان‌گیری آسان

مثال‌های کاربردی

1. ریتم تکنو پایه

setCps(130/60/4) // BPM 130 stack( sound("bd*4"), // کیک 4/4 sound("~ sd ~ sd"), // اسنر آف‌بیت sound("hh*16").gain(0.3), // های‌هت سریع sound("~ ~ ~ cp").room(0.5) // کلپ با ریورب )

این الگو یک ریتم تکنو کلاسیک با BPM 130 می‌سازد.

2. بیس‌لاین اسید

note("c2 c3 eb2 g2") .sound("sawtooth") .lpf(sine.range(200,2000).slow(8)) .lpq(10) .delay(0.125) .room(0.2)

بیس‌لاین با فیلتر متحرک که صدای اسید تولید می‌کند.

3. درام اند بیس

setCps(175/60/4) // BPM 175 stack( sound("bd ~ ~ bd ~ ~ bd ~"), // کیک سینکوپ sound("~ ~ sd ~ ~ ~ sd ~"), // اسنر sound("hh*8").gain("{0.2 0.4}"), // های‌هت داینامیک note("c1 ~ ~ c1 ~ c1 ~ ~") // ساب بیس .sound("sine").gain(0.8) ).jux(rev)

پترن درام اند بیس با ریتم شکسته و بیس عمیق.

4. آمبینت

note("") .sound("triangle") .slow(4) .room(0.9) .delay(0.5) .gain(0.3) .lpf(600)

پد آمبینت آرام با افکت‌های فضایی.

5. گلیچ هاپ

stack( sound("bd*2").sometimes(x => x.speed(2)), sound("{~ sd*2 ~ cp}").fast(2), sound("hh*8").sometimesBy(0.3, x => x.speed(-1)), sound("glitch*4").gain(0.2).speed("{1 2 0.5}") ).every(4, x => x.rev())

ریتم گلیچ با تغییرات تصادفی و معکوس شدن هر 4 سیکل.

6. پلی‌ریتم آفریقایی

stack( sound("bd").euclid(3,8), // الگوی اقلیدسی 3 در 8 sound("sd").euclid(5,8), // الگوی اقلیدسی 5 در 8 sound("hh").euclid(7,8).gain(0.3), // الگوی اقلیدسی 7 در 8 sound("cp").euclid(4,8).delay(0.125) // الگوی اقلیدسی 4 در 8 )

استفاده از ریتم‌های اقلیدسی برای ساخت پلی‌ریتم پیچیده.

میانبرهای صفحه‌کلید

میانبرهای اصلی

  • Ctrl + Enter - اجرای کد
  • Ctrl + . - توقف صدا
  • Ctrl + Z - Undo
  • Ctrl + Y - Redo

ویرایش کد

  • Tab - تورفتگی
  • Shift + Tab - حذف تورفتگی
  • Ctrl + / - کامنت

نکات و ترفندهای حرفه‌ای

از setCps() برای تنظیم BPM استفاده کنید. فرمول: BPM / 60 / 4
از every(n, fn) برای اعمال تغییرات دوره‌ای استفاده کنید. مثلا: sound("bd*4").every(4, x => x.rev())
از sometimes() و sometimesBy(0.5) برای تغییرات تصادفی استفاده کنید.
از stack() برای لایه‌بندی و cat() برای توالی استفاده کنید.
از .jux(fn) برای اعمال تابع روی کانال راست استفاده کنید. مثل: .jux(rev)
همیشه از hush() قبل از کد جدید استفاده کنید تا صداهای قبلی متوقف شوند.
